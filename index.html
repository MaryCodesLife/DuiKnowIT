What is a DuiKnowIT device?


Poverty, natural disasters, human right violations, and crimes are few challenges that Filipinos have been facing for years now. Even though there is an increasing number of internet and cellular providers, there are still areas that do not have cellular networks.

We, DuiKnowIT team, created a functional device even in a poor signal strength environment that would provide communication and tracker during and immediately after the crime and catastrophe which are vital components of response and recovery.

By using a smartphone that has DuiKnowIt application installed, we can connect to each device via Bluetooth where we can communicate and locate with other people having the same setup. The DuiKnowIT device has different water-resistant and weatherproof cases (Transparent Enclosed Case and 3D-printed Case).

The intended use of the application and device is for long range communication in off-grid areas. Another possible application is for personal use in the event that there is no cellular and internet coverage.
